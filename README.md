# My Environment Setup

My personal development environment configuration and dotfiles.

## Stack

- **Shell**: Zsh
- **Terminal**: Ghostty
- **Editor**: Neovim
- **Multiplexer**: Tmux
- **WM**: Hyprland

## What's included

- Zsh configuration
- Tmux configuration
- Neovim config
- Ghostty terminal config
- Hyprland keybindings and monitor setup
- Custom environment setup

## Sync script

Run `./copy-configs.sh` to sync the latest configurations from the system.
